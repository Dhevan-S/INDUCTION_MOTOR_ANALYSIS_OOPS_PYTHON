# Induction Motor Performance Analysis using OOP

This project presents a Python-based application for analyzing the performance characteristics of an induction motor using Object-Oriented Programming principles.

The system models the motor by computing key electrical parameters such as synchronous speed, slip, electromagnetic torque, output power, and efficiency based on user-provided inputs. It incorporates fundamental electrical engineering formulas and simulates real-time motor behavior.

The application is designed with a modular OOP structure, including a base Motor class and a derived InductionMotor class. Additional components handle performance classification and graphical visualization, ensuring clear separation of logic and maintainability.

The project also includes:
- Classification of motor operating regions based on slip and rotor speed  
- Visualization of performance curves such as torque-speed, slip-speed, and efficiency vs output power  
- Automated generation of a detailed report summarizing input parameters and computed results  

This project demonstrates the integration of core electrical engineering concepts with software design, providing a practical tool for understanding and analyzing induction motor performance.
